# Quotes and nested lists

> A useful markdown workbench should keep quoted content legible
> even while the surrounding sections continue to grow.

## Review notes

1. Primary goals
   - Make the selected example obvious.
   - Keep the toolbar quiet and efficient.
   - Preserve long-form reading comfort.
2. Editor handoff
   - Use a neutral canvas.
   - Keep the sidebar lighter than the preview area.

> Follow-up
>
> - Nested bullets should remain grouped.
> - Ordered steps should not jump during streaming.
